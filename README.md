# Mini Pinball

A lightweight GitHub Pages-ready pinball mini game built with plain HTML, CSS, and JavaScript.

## Features
- Canvas playfield
- Moving pinball with gravity
- Wall bounce
- Left / right flippers
- Arrow key controls
- Score system
- Game over on drain
- Press `R` to restart

## Run locally

```bash
cd pinball-game && python3 -m http.server 8000
```

Open <http://localhost:8000>

## GitHub Pages
Set GitHub Pages to:
- Source: Deploy from a branch
- Branch: `main`
- Folder: `/(root)`

Then open:

```text
https://<your-account>.github.io/pinball-game/
```
